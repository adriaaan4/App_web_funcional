# Evidencia GA4-220501096-AA1-EV01: API del Proyecto

Requisitos:
- Tener **Node.js** instalado.
- Tener **MongoDB** instalado y en ejecución.

Instalación:
Clonar el repositorio o descargar el proyecto.
Dentro de la carpeta del proyecto, ejecutar:

**npm install** (No se subió node_modules)

Iniciar servidor:
**npm run dev**
